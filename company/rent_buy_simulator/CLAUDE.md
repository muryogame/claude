# rent_buy_simulator — 家賃vs購入 コスト比較シミュレーター

## 概要
賃貸と購入どちらが得かを計算する不動産コストシミュレーター。
純静的HTML/CSS/JS（サーバー不要）。Chart.js CDN を使用。

## ファイル構成
```
rent_buy_simulator/
├── index.html   # メインHTML（3タブ構成）
├── style.css    # スタイル
└── app.js       # 計算ロジック + Chart.js グラフ
```

## 機能
| タブ | 内容 |
|------|------|
| 家賃vs購入 比較 | 比較期間の累積コストを折れ線グラフで比較、損益分岐点を自動計算 |
| 住み替えコスト | 賃貸→賃貸/賃貸→購入/持ち家売却の引越し費用内訳を計算 |
| ローン返済計算 | 元利均等/元金均等シミュレーション、返済スケジュール表 |
