# health_tools — 処方薬 飲み合わせチェッカー ＋ 睡眠スコア記録

## 概要
処方薬・市販薬の飲み合わせチェックと、睡眠スコアの毎日記録・グラフ可視化ツール。
純静的HTML/CSS/JS（サーバー不要）。データはLocalStorageに保存（外部送信なし）。

## ファイル構成
```
health_tools/
├── index.html   # メインHTML（2タブ構成）
├── style.css    # スタイル（グリーン基調）
└── app.js       # 薬DBチェック + 睡眠記録 + Chart.js グラフ
```

## 機能
| タブ | 内容 |
|------|------|
| 飲み合わせチェック | 20種の薬カテゴリ × 18種の相互作用DB、危険/警告/情報の3段階アラート |
| 睡眠スコア記録 | 就寝・起床時刻・スコアを毎日記録、30日分グラフ表示（LocalStorage永続化） |

## 薬データベース（app.js 内 DRUG_DB）
- ワーファリン、アスピリン、NSAIDs、降圧薬、スタチン、睡眠薬、SSRIなど20種類
- 相互作用（INTERACTIONS 配列）: 出血リスク、グレープフルーツ、アルコール等18パターン

## データ永続化
- LocalStorage キー: `sleep_records_v1`
- サーバー送信なし・ログイン不要
